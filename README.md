# uk-road-accident-analysis-dashboard🚦 UK Road Accident Analysis Dashboard
🔍 Project Overview
This Power BI dashboard provides a comprehensive analysis of UK road accident data to uncover patterns in accident frequency, casualty distribution, vehicle involvement, and district-level risk trends.
The goal of this project is to support data-driven decision-making for improving public road safety.

🎯 Business Objectives
Identify accident-prone districts
Analyze casualty distribution and severity trends
Study quarterly accident patterns
Examine vehicle-type involvement in accidents
Evaluate the impact of road surface conditions on accident occurrence
🛠 Tools & Technologies Used
Power BI Desktop
DAX (Calculated Measures & KPIs)
Power Query (Data Cleaning & Transformation)
Data Modeling & Relationship Management
📊 Key Performance Indicators (KPIs)
Metric	Value
Total Casualties	25K
Average Speed Limit	31.09
📈 Key Insights & Findings
📍 Westminster recorded the highest number of accidents.
🚗 Cars are involved in the majority of reported accidents.
🚦 T-junctions and crossroads are identified as high-risk zones.
📅 Accident frequency peaks in Q4, indicating seasonal variation.
🌤 Most accidents occur under dry road conditions, highlighting driver behavior as a key factor.
